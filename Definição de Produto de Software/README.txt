
Você pode usar o latex, e esse template, diretamente pelo overleaf: https://www.overleaf.com

---

Caso queira utilizar diretamente da sua máquina você deve:

- Instalar o miktex (Windows) ou texlive (Linux): 
https://miktex.org/
https://www.tug.org/texlive/

- Instalar um editor de Latex. Por exemplo, Texmaker:
https://www.xm1math.net/texmaker/

Basta abrir o template no editor e usar.

---

No Texmaker você deve mudar as configurações de build. Para isso:

Siga para o menu: Options -> Configure Texmaker -> Quick Build -> Selecione a segunda opção: PdfLateX + Bib(la)tex + PdfLateX (x2) + View Pdf.

---

Além disso, você pode configurar o Editor para fazer verificações ortográficas no texto em português BR. Para isso, você deve baixar o dicionário do libreoffice (https://pt-br.libreoffice.org/projetos/vero) e fazer referencia do caminho do arquivo "pt_BR.dic" no menu: 

Options -> Configure Texmaker -> Editor -> Spelling dictionary

Para facilitar, este dicionário está no diretório "corretor_pt_BR".
